Privacy Policy

Last Updated: July 15, 2025

Overview

This site uses Google OAuth to allow users to authenticate with their Google account for the purpose of automating content generation workflows (e.g., posting videos to YouTube).

Data Collection

We do not collect, store, or share any personal data beyond what is required to complete authorized actions through the Google APIs. OAuth tokens are used solely to authenticate with Google services (e.g., YouTube API) and are never shared or sold.

How Data is Used

  • Google account information is used only for authentication purposes.
  • YouTube access is used to publish and manage videos you explicitly generate and approve.
  • No emails, contacts, or private data are accessed or stored.

Data Security

All tokens and API access are handled securely on a private self-hosted server and are never exposed to third parties. Access is limited to the authenticated user only.

User Consent

By using the Google Sign-In feature, you consent to the use of Google OAuth solely for the purposes described above.

Contact

If you have any questions or concerns about this privacy policy, please contact the system administrator at ktsmith.kcp@gmail.com.

This app is non-commercial and used for private, personal automation purposes only.
